# Recipe Manager (Console Application)

## Project Description
Recipe Manager is a console-based application built with TypeScript that allows users to manage Open Source cooking recipes. Users can create, edit, delete, view, and Export/Import recipes through JSON file format. All data is stored locally using Sqlite database.

---

## Target Users / Use Cases
- Home cooks organizing personal recipes
- Users who want a lightweight, offline recipe manager

---

## Problem Statement

### Problem
Recipes are often scattered across notebooks or messages, making them difficult to manage and update.

### Solution
Recipe Manager provides a centralized and digital way to store, update, and share recipes using a simple console application.

---

## Key Features
1. Create recipe
2. Edit recipe
3. Delete recipe
4. View recipes
5. View recipe steps
6. Export/Import Recipes

---

## Technology Stack
- Language: TypeScript
- Runtime: Node.js
